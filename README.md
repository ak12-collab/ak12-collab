### 📁 Featured Projects

#### 💼 [Corporate Budget Variance & Executive Briefing System](https://github.com/ak12-collab/Budget-variance-executive-system)
> Python · Excel · Power BI · PDF

End-to-end FP&A reporting system replicating enterprise finance workflows.
Built a synthetic 288-row dataset with engineered variance patterns, an Excel variance 
model with waterfall charts, a 2-page Power BI dashboard with DAX measures and heat map, 
and a CFO one-page summary PDF.

**Key Insight:** *Marketing overspent by 34% in Q3 while Operations underspent by 18% — 
the net variance masked a structural reallocation problem invisible in aggregate reporting.*
